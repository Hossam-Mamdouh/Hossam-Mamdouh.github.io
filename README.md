# Hossam Mamdouh Portfolio Website

**Live Preview:** [https://hossam-mamdouh.github.io/](https://hossam-mamdouh.github.io/)

Personal portfolio website for Hossam Mamdouh — Embedded Systems Engineer, IoT Developer, and PCB Designer based in Cairo, Egypt. Built with plain HTML, CSS, and JavaScript; no frameworks or build tools required.

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home page — hero section, skills summary, and key metrics |
| `about.html` | Profile, background, education, and toolchain overview |
| `services.html` | Service offerings: Firmware Development, PCB Design, IoT Development, Circuit Design & Simulation |
| `portfolio.html` | Project gallery — embedded, IoT, robotics, and PCB projects |
| `contact.html` | Contact links and freelance platform profiles |

## Project Highlights

- Water Level Monitoring and Control PCB (ESP32)
- IoT-Based ECG Monitoring System (ESP32 + AD8232)
- Medication Monitoring System / Smart Pill Box (Arduino Uno WiFi)
- Hand Gesture Controlled Car (Arduino Pro Mini + RF)
- Automated Solar Panel Cleaner System (Arduino Uno + IR)
- Smart Irrigation System
- Automated Garage System (Arduino Mega)
- MicroSD Audio Board
- Arduino Uno with ESP-12E WiFi Integration
- Stepper Motor Conveyor Arm
- Terminal Tedium — Audio Mixer and Effects Board
- 5V to 12V Voltage Booster, LiPo Battery Charger, DS3231 RTC Module (custom PCBs)
- ESP32 Temperature Controller

## Tech Stack

- **Languages:** HTML5, CSS3, vanilla JavaScript (ES6+)
- **Fonts:** Space Grotesk, Manrope (Google Fonts)
- **Microcontrollers covered:** AVR, PIC, 8051, STM32 / ARM Cortex-M, MSP430, ESP32, Arduino
- **PCB tools covered:** Altium Designer, KiCad, EasyEDA, Eagle

## File Structure

```
index.html

services.html

contact.html
```

## Run

Open `index.html` in any modern browser to start from the home page. No build step or server required.
