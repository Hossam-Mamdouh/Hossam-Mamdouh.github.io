# Hossam Mamdouh Portfolio Website

A personalized multi-page portfolio website built with plain HTML, CSS, and JavaScript.

## Pages

- `index.html` home page
- `about.html` profile and background
- `services.html` service offerings
- `portfolio.html` selected work and project categories
- `contact.html` contact pathways and platform links

## Shared Files

- `styles.css` shared design system and responsive layout
- `script.js` mobile navigation and scroll-reveal behavior

## Content Sources Used

This version was personalized using the public profile links shared by the user:

- LinkedIn profile URL
- Upwork profile URL
- Freelancer profile URL
- GitHub Pages personal site URL

Some details were directly visible in public search results, especially from the Freelancer and Upwork pages.
Where profile pages were not fully accessible, the site copy uses conservative summaries based on the public snippets that were available.

## Run

Open `index.html` in a browser to start from the home page.
